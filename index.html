Tentang Saya.

Menerjemahkan Data, Membangun Solusi, dan Menciptakan Dampak.

Ketertarikan utama saya terletak pada eksplorasi dan analisis data. Saya sangat menikmati keseluruhan prosesnya—mulai dari pre-processing data mentah, mengevaluasi model, hingga menemukan insight yang berharga. Bagi saya, kendala di tengah proses analisis bukanlah hambatan, melainkan ruang untuk berpikir kritis dan memecahkan masalah.

Dalam perjalanan saya membangun karir sebagai Data Scientist/Analyst, saya juga memiliki hasrat yang kuat dalam pengembangan antarmuka visual dan produk cipta guna. Saya menikmati proses merancang model AI yang bisa langsung diterapkan di dunia nyata, serta membangun antarmuka web yang interaktif.

Saya percaya teknologi bekerja paling baik ketika ia inklusif. Ke depannya, saya memiliki visi untuk mengintegrasikan analitik data dan robotika guna menciptakan teknologi tepat guna, khususnya bagi penyandang disabilitas.

Fokus Riset.

Eksplorasi teknologi untuk solusi agrikultur dan komputasi cerdas.

Riset ini berfokus pada pemanfaatan Generative Adversarial Networks (GAN) untuk menyintesis data citra agrikultur. Tujuannya adalah memecahkan masalah kelangkaan data visual spesifik guna melatih model prediksi hasil panen kelapa sawit agar jauh lebih akurat dan efisien.

Penelitian ini mengevaluasi bagaimana variasi tingkat detail spasial (GSD) dari tangkapan drone (UAV) memengaruhi kinerja model prediksi AI. Hasil dari riset ini diharapkan dapat menjadi standar operasional dalam pengambilan data udara untuk perkebunan presisi.
